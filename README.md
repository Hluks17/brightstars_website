Bright Stars is a 3 months project which was split into 3 parts layout styling and user experience.
below are the parts and details of the project

Part one
Bright Stars Tutoring is a Non-Governmental Organization (NGO) dedicated to helping students succeed academically by providing tutoring classes and mentorship. The website serves as an online platform where learners, parents, and the community can learn more about our programs, services, and ways to get in touch.
The site was created using basic HTML tags only (no CSS stylesheet). This makes it lightweight, beginner-friendly, and easy to run on any browser.
Pages Included
The website consists of four main pages:
1.
Home Page (index.html) o Introduction to Bright Stars Tutoring. o Overview of our mission and vision.
o
Display of pictures showing students in action.
2.
About Us (about.html) o Detailed information about who we are. o Our goals and values.
o
Explanation of why we exist as an NGO.
3.
Services (services.html) o List of tutoring programs (Maths, English, Science, Exam Prep, etc.). o Extra features like mentorship and career guidance.
o
Picture gallery with 6 images of students in class.
4.
Contact Us (contact.html) o Information on how to reach us. o Links to WhatsApp, Facebook, and Instagram. o Email and phone contact details.
Design Choices
•
Background Color: Dark Bold Blue (#00008B) to give the site a strong academic feel.
•
Header & Footer Color: Dark Goldenrod (a bold gold shade) for elegance and trustworthiness.
•
Font Color: Gold for readability and visual contrast.
•
Images: Six placeholders (with filenames 123.jpg) were used to represent tutoring sessions.
How It Was Built
•
HTML was used to build the structure of all pages.
•
The <table> tag was used to create headers, navigation menus, and footers.
•
The <img> tag was used to insert images.
•
Inline HTML attributes like bgcolor, text, and font were used instead of CSS for styling.
Part Two Enhancements & Styling
In this phase of development, I focused on improving the design, interactivity, and user experience of the Bright Stars Tutoring website. Below are the features I added and their benefits:
1. CSS Styling
•
Introduced a separate CSS stylesheet (style.css) for a clean structure.
•
Styled headers, navigation, and footer with dark blue and gold, ensuring consistent branding.
•
Used responsive grid cards for images to make the website look professional and adaptable to all screen sizes.
Benefit: Separating design from content makes the website easier to maintain and improves the overall professional look.
2. Sticky Navigation Bar
•
Made the navigation bar sticky, meaning it stays at the top when scrolling.
•
Added smooth scrolling across the site.
Benefit: Users can always access the menu without scrolling back up, improving usability and navigation.
3. Card Layout for Images
•
Placed tutoring pictures into card components with borders, shadows, and hover effects.
•
Each card now has room for a caption or description to explain the activity.
Benefit: This makes the gallery more engaging, visually appealing, and informative.
4. Buttons with Hover Effects
•
Added gold action buttons across sections (e.g., “Learn More” / “Join Now”).
•
Buttons change color and slightly scale up when hovered.
Benefit: Clear call-to-action elements guide visitors to take the next step while making the website interactive.
5. Upgraded Contact Form
•
Redesigned the Get in Touch form with larger, styled input fields.
•
Styled the Submit button with hover effects.
Benefit: Bigger and clearer input fields improve usability, especially on mobile. The professional styling increases trust and encourages users to reach out.
6. Social Media Integration
•
Added social media icons (Facebook, Twitter, Instagram) using Font Awesome.
•
Styled them to match the gold-and-dark-blue theme with hover effects.
Benefit: Makes it easy for users to connect with Bright Stars Tutoring on social platforms, boosting visibility and engagement.
7. Responsiveness
• Applied media queries so the site adapts to different devices (desktop, tablet, and mobile).
Benefit: Students and parents can comfortably access the site from their phones, which is crucial for accessibility.
Overall Impact
With these updates, the website now feels modern, interactive, and user-friendly. The design uses a strong visual identity (dark blue + gold), while added features like cards, buttons, and sticky navigation make it professional and engaging.
Part Three Improvements
For Part Three of the project, several new features and improvements were implemented to enhance the overall functionality, interactivity, and user experience of the website. These enhancements focused on making the site more user-friendly, visually appealing, and accessible across devices.
1.
Scroll-to-Top Button A scroll-to-top button was added, allowing users to easily return to the top of any page without excessive scrolling. This improves navigation efficiency, especially on long pages, and reduces user frustration.
2.
FAQ Accordion Section The FAQ section was updated to be interactive, with questions that expand to reveal answers when clicked. This provides users with quick access to common information, reduces clutter on the page, and improves readability.
3.
Mobile Responsiveness The entire website was made mobile-friendly using media queries in the CSS. All sections, including the hero banner, cards, forms, and navigation, now adapt to different screen sizes. This ensures a seamless experience for users on phones, tablets, and desktop devices, which is essential for accessibility and modern web standards.
4. Image Gallery / Lighthouse Gallery
An interactive gallery feature was implemented to showcase images in a visually appealing manner. Users can click images to view them in full-screen mode. This enhances the visual experience, engages users, and allows them to focus on content without distractions.
5.Embedded Map A map was added to the Contact page, allowing users to easily locate the tutoring center. This feature provides practical value, helps users plan visits, and improves trust and reliability by clearly showing the location.
